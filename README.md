# LogicaDE


ГОТОВЫ:

- GUI
- Логотип
- Сеттинг(Сюжет, некоторые из головоломок)
- фоны/заставки
- музыка(SFX)
- Система сохранения сессий
- Поддержка нескольких сессий
- Сборка всех скриптов и запуск через один файл

В ПРОЦЕССЕ:

- GUI(всё готово, осталась только работа с текстурами)
- головоломки
- кампания/свободный режим

Общий сеттинг(subject to change):

Игра происходит в таинственном подземелье куда гг был заброшен странной магией. По ходу прохождения квеста будут открываться всё новые головоломки.
Визуал(фоны, заставки) выполнены в стиле темного фэнтези.
Планируется создание 25 уровней с головоломками разной сложности которые помогут в развитии логики.
