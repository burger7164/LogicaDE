Общий сеттинг:

Игра происходит в таинственном подземелье куда гг был заброшен странной магией. По ходу прохождения квеста будут открываться всё новые головоломки.
Визуал(фоны, заставки) выполнены в стиле темного фэнтези.
В подземелье 25 уровней с головоломками разной сложности которые помогут в развитии логики.
Также присутсвует открытый режим..
